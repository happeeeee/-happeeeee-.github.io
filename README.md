# 筑云小阁 - GitHub Pages 静态站点模板

这是为 “筑云小阁”（项目：happeeeee/happeeeee.github.io）准备的静态站点模板，
支持通过 `admin.html` 在浏览器中上传文章与图片、修改站点配置、开启 Giscus 评论以及客户端文章加密功能。

快速说明
- 打开网站首页: https://happeeeee.github.io/happeeeee.github.io/（启用 GitHub Pages 后）
- 后台管理页面: https://happeeeee.github.io/happeeeee.github.io/admin.html
- 在 admin.html 中粘贴你的 GitHub Personal Access Token (PAT) 并登录后即可上传文章、图片或保存 config.json。

重要：
- 请不要把你的 PAT 写入仓库文件。生成并粘贴到 admin.html 的登录框中即可。
- 若需要 Giscus 评论完整功能，请访问 https://giscus.app/ 生成对应配置并填入 config.json（我在说明里会写如何操作）。

文件清单（本次准备）
- index.html
- admin.html
- assets/styles.css
- assets/main.js
- assets/admin.js
- config.json (已填你的信息)
- articles/nihao-shijie.md （示例文章）
- articles/index.json （示例文章索引）

下一步请确认这些文件内容，确认无误后我会告诉你如何在 GitHub 页面上一步步上传（按你选的交付方式 A）。
